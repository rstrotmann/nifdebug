# nifdebug

<!-- badges: start -->

<!-- badges: end -->

nifdebug provides functions to troubleshoot Nonmem Input Format (NIF)
data sets generated with the nif package.

## Installation

You can install the development version of nifdebug like so:

``` r
devtools::install_github("rstrotmann/nifdebug)
```
